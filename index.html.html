        <li><strong>Why This Problem Needs to Be Solved (Impact & Relevance):</strong> Over 2 million new cases are diagnosed annually (WHO, 2023), with mortality rates highest in low-resource settings where awareness and early detection are scarce. Solving this reduces preventable deaths, improves quality of life, and addresses health inequities, aligning with global health goals.</li>
        <li><strong>Existing Solutions and Their Limitations:</strong> Mammograms and clinical exams exist but are costly, inaccessible in rural areas, and often unavailable to the visually impaired or differently-abled. Self-examination lacks accuracy and awareness campaigns miss diverse populations due to language and technology barriers.</li>

    <div class="content">

        <li><strong>How AI is Utilized:</strong> Employs machine learning for image-based self-diagnosis (e.g., analyzing skin changes), NLP for multilingual support and chatbots, and text-to-speech for accessibility, enabling early risk detection and education.</li>
        <li><strong>Expected Outcomes and Benefits:</strong> Early detection increases survival rates by up to 90% (American Cancer Society), reduces treatment costs, and empowers users with knowledge and privacy, breaking socio-economic barriers.</li>

<script>
  const slides = document.querySelectorAll('.slide');
  const indicators = document.querySelectorAll('.indicator');
  let currentSlide = 0;
  const prevBtn = document.getElementById('prevBtn');
  const nextBtn = document.getElementById('nextBtn');

  function showSlide(index) {
    slides.forEach((slide, i) => {
      slide.classList.toggle('active', i === index);
    });
    indicators.forEach((indicator, i) => {
      indicator.classList.toggle('active', i === index);
    });
    prevBtn.disabled = index === 0;
    nextBtn.disabled = index === slides.length - 1;
  }

  function nextSlide() {
    if (currentSlide < slides.length - 1) {
      currentSlide++;
      showSlide(currentSlide);
    }
  }

  function prevSlide() {
    if (currentSlide > 0) {
      currentSlide--;
      showSlide(currentSlide);
    }
  }

  function goToSlide(index) {
    currentSlide = index;
    showSlide(currentSlide);
  }

  function toggleFullscreen() {
    if (!document.fullscreenElement) {
      document.documentElement.requestFullscreen();
    } else if (document.exitFullscreen) {
      document.exitFullscreen();
    }
  }

  // Initialize the first slide
  showSlide(currentSlide);

  // Keyboard navigation
  document.addEventListener('keydown', (e) => {
    if (e.key === 'ArrowRight') nextSlide();
    else if (e.key === 'ArrowLeft') prevSlide();
    else if (e.key === 'f') toggleFullscreen();
  });
</script>
